# iwebhooks

Webhooks

## Quick Start

### 1. Start directly:

```bash
pip install -r requirements.txt

python app.py
```

### 2. Start by docker-compose

```bash
docker-compose up -d iwebhook
```

### 3. Deploy in kubernetes by helm

```bash
helm install iwebhook 
```
